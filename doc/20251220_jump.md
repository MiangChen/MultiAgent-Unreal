# Jump 技能设计

## 需求

1. 空格键触发选中 Agent 跳跃
2. 导航中卡住时自动调用跳跃技能（统一逻辑）

## 当前状态

RobotDog 已有自动跳跃逻辑 (`CheckStuckAndJump`)：
- 卡住时自动检测并跳跃
- `LaunchCharacter(前200, 上600)`
- 冷却 1 秒

**问题**: 跳跃逻辑写死在 RobotDog 中，需要统一到基类。

## 设计方案

### 统一跳跃接口 (MACharacter 基类)

```cpp
// MACharacter.h
protected:
    /** 跳跃冷却 */
    float JumpCooldown = 0.f;
    
    /** 卡住检测时间 */
    float StuckTime = 0.f;
    float StuckThreshold = 0.5f;

public:
    /** 是否能跳跃 */
    UFUNCTION(BlueprintCallable)
    virtual bool CanPerformJump() const;

    /** 执行跳跃 (手动或自动触发) */
    UFUNCTION(BlueprintCallable)
    virtual void PerformJump();

protected:
    /** 检测卡住并自动跳跃 (在 Tick 中调用) */
    virtual void CheckStuckAndAutoJump(float DeltaTime);
```

### MACharacter 基类实现

```cpp
// MACharacter.cpp
bool AMACharacter::CanPerformJump() const
{
    // 在地面上 + 冷却结束
    return GetCharacterMovement()->IsMovingOnGround() && JumpCooldown <= 0.f;
}

void AMACharacter::PerformJump()
{
    if (!CanPerformJump()) return;
    
    // 向前+向上跳跃
    FVector JumpImpulse = GetActorForwardVector() * 200.f + FVector(0.f, 0.f, 500.f);
    LaunchCharacter(JumpImpulse, false, true);
    
    JumpCooldown = 1.0f;
    StuckTime = 0.f;
    
    UE_LOG(LogTemp, Log, TEXT("[%s] PerformJump"), *GetName());
}

void AMACharacter::CheckStuckAndAutoJump(float DeltaTime)
{
    // 更新冷却
    if (JumpCooldown > 0.f)
    {
        JumpCooldown -= DeltaTime;
    }
    
    // 只有在移动中才检测
    if (!bIsMoving) 
    {
        StuckTime = 0.f;
        return;
    }
    
    // 速度很低 = 可能卡住
    if (GetVelocity().Size() < 10.f)
    {
        StuckTime += DeltaTime;
        
        if (StuckTime > StuckThreshold)
        {
            // 前方射线检测静态障碍物
            FHitResult Hit;
            FVector Start = GetActorLocation();
            FVector End = Start + GetActorForwardVector() * 80.f;
            
            FCollisionQueryParams Params;
            Params.AddIgnoredActor(this);
            
            if (GetWorld()->LineTraceSingleByChannel(Hit, Start, End, ECC_WorldStatic, Params))
            {
                // 调用统一的跳跃接口
                PerformJump();
            }
            else
            {
                StuckTime = 0.f;  // 被其他 Agent 挡住，不跳
            }
        }
    }
    else
    {
        StuckTime = 0.f;
    }
}
```

### 各 Agent 类型

| Agent 类型 | CanPerformJump | 说明 |
|-----------|----------------|------|
| Human | ✅ 基类默认 | 标准跳跃 |
| RobotDog | ✅ 基类默认 | 移除原有 CheckStuckAndJump |
| Drone | ❌ 重写返回 false | 飞行单位不跳跃 |

```cpp
// MADroneCharacter.cpp
bool AMADroneCharacter::CanPerformJump() const
{
    return false;  // 无人机不跳跃
}

void AMADroneCharacter::CheckStuckAndAutoJump(float DeltaTime)
{
    // 无人机不需要跳跃检测
}
```

### 输入绑定 (空格键)

```cpp
// MAInputActions.h
UPROPERTY()
UInputAction* IA_Jump;

// MAInputActions.cpp
IA_Jump = NewObject<UInputAction>(this);
IA_Jump->ValueType = EInputActionValueType::Boolean;
IMC_RTS->MapKey(IA_Jump, EKeys::SpaceBar);

// MAPlayerController.cpp
void AMAPlayerController::OnJumpPressed()
{
    if (UMASelectionManager* SM = GetWorld()->GetSubsystem<UMASelectionManager>())
    {
        for (AMACharacter* Agent : SM->GetSelectedAgents())
        {
            if (Agent)
            {
                Agent->PerformJump();
            }
        }
    }
}
```

## 实现步骤

1. MACharacter 基类添加 `CanPerformJump()`, `PerformJump()`, `CheckStuckAndAutoJump()`
2. MACharacter::Tick 中调用 `CheckStuckAndAutoJump()`
3. RobotDog 移除原有的 `CheckStuckAndJump()` (使用基类)
4. Drone 重写 `CanPerformJump()` 返回 false
5. MAInputActions 添加 `IA_Jump`
6. MAPlayerController 添加 `OnJumpPressed()`
7. 编译测试

## 调用关系

```
手动触发:  空格键 → PlayerController::OnJumpPressed() → Agent::PerformJump()
自动触发:  Tick → CheckStuckAndAutoJump() → PerformJump()
                                ↓
                        统一的跳跃逻辑
```
