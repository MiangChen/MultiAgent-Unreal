## 4. 智能体数量 - 要求 ≥30 个

### 当前架构

- `UMAActorSubsystem` 管理所有 Character
- 支持动态生成和销毁
- 每个 Character 独立的 StateTree + ASC

### 性能考虑

| 数量级 | 方案 | 状态 |
|--------|------|------|
| 1-30 | 标准 Character + StateTree | ✅ 已实现 |
| 30-100 | 需要优化 StateTree Tick | 🔄 待优化 |
| 100+ | 考虑 Mass AI 框架 | 📋 已规划 |

**状态**: ✅ 架构支持，需要性能测试

## 5. 组合任务 - 要求 ≥200 个

任务组合数 = 任务类型 × 智能体类型 × 环境目标
