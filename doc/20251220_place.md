# Agent 运行时部署位置选择方案

> 日期: 2025-12-20
> 需求: 在运行时让用户手动选择 Agent 的部署位置
> 状态: **已实现方案三 (拖拽框选)**

## 当前架构

```
MASetupWidget (配置类型/数量)
    ↓
MASetupHUD::OnStartSimulation() (保存到 GameInstance)
    ↓
切换地图
    ↓
MAGameMode::SpawnAgentsFromSetupConfig()
    ↓
MAPlayerController::EnterDeploymentMode() (进入部署模式)
    ↓
用户拖拽框选区域
    ↓
Agent 在框选区域内生成
```

---

## 已实现：方案三 - 拖拽框选区域 (Drag-to-Deploy)

### 流程

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│  SetupWidget    │ ──► │  DeploymentMode  │ ──► │  AgentManager   │
│  (选择类型)      │     │  (框选区域)       │     │  (区域内生成)   │
└─────────────────┘     └──────────────────┘     └─────────────────┘
```

### 使用方法

1. 在 Setup 界面配置 Agent 类型和数量
2. 点击 "Start Simulation" 进入地图
3. 自动进入 **Deployment 模式** (蓝色框选)
4. 拖拽鼠标框选一个区域，Agent 会在该区域内网格分布生成
5. 每种类型需要单独框选一次
6. 全部部署完成后自动退出部署模式

#### 2. 进入部署模式 (MAPlayerController.cpp)

```cpp
void AMAPlayerController::EnterDeploymentMode(const TArray<FMAPendingDeployment>& Deployments)
{
    PreviousMouseMode = CurrentMouseMode;
    CurrentMouseMode = EMAMouseMode::Deployment;
    PendingDeployments = Deployments;
    CurrentDeploymentIndex = 0;
    DeployedCount = 0;
    
    // 设置输入模式
    bShowMouseCursor = true;
    SetIgnoreLookInput(true);
}
```

#### 3. 框选放置 (MAPlayerController.cpp)

```cpp
void AMAPlayerController::OnDeploymentLeftClickReleased()
{
    // 获取框选区域
    FVector2D Start = SelectionManager->GetBoxSelectStart();
    FVector2D End = SelectionManager->GetBoxSelectEnd();
    
    // 计算生成点 (网格分布)
    TArray<FVector> SpawnPoints = ProjectSelectionBoxToWorld(Start, End, Count);
    
    // 生成 Agent
    for (const FVector& Point : SpawnPoints)
    {
        AgentManager->SpawnAgentByType(AgentType, Point, FRotator::ZeroRotator, false);
    }
}
```

#### 4. 屏幕坐标转世界坐标 (MAPlayerController.cpp)

```cpp
TArray<FVector> AMAPlayerController::ProjectSelectionBoxToWorld(FVector2D Start, FVector2D End, int32 Count)
{
    // 将屏幕四角投影到世界
    FVector WorldCorners[4];
    DeprojectScreenPositionToWorld(...);
    
    // 投影到地面
    for (int32 i = 0; i < 4; ++i)
        WorldCorners[i] = ProjectToGround(WorldCorners[i]);
    
    // 网格分布 + 随机偏移
    int32 Cols = FMath::CeilToInt(FMath::Sqrt((float)Count));
    int32 Rows = FMath::CeilToInt((float)Count / Cols);
    
    // 双线性插值生成点
    for (Row, Col)
        Point = BiLerp(WorldCorners, U, V);
}
```

#### 5. GameMode 触发部署模式 (MAGameMode.cpp)

```cpp
void AMAGameMode::SpawnAgentsFromSetupConfig(UMAGameInstance* GameInstance)
{
    // 构建待部署列表
    TArray<FMAPendingDeployment> Deployments;
    for (const auto& Pair : GameInstance->SetupAgentConfigs)
        Deployments.Add(FMAPendingDeployment(Pair.Key, Pair.Value));
    
    // 进入部署模式
    AMAPlayerController* MAPC = Cast<AMAPlayerController>(PC);
    MAPC->EnterDeploymentMode(Deployments);
}
```
