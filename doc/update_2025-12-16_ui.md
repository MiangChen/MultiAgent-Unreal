# 更新日志: 2025-12-16 Setup UI 系统

## 概述

本次更新实现了仿真启动前的 Setup 阶段 UI 系统，允许用户在进入仿真前配置场景和 Agent 小队组成。

## 核心功能

| 功能 | 说明 |
|------|------|
| 场景选择 | 下拉菜单选择仿真场景 (CyberCity, OldTown, Factory, Forest, DesertBase) |
| Agent 类型选择 | 下拉菜单选择 Agent 类型 (DronePhantom4, DroneInspire2, RobotDog, Human) |
| 数量配置 | 通过 [+] [-] 按钮调整每种 Agent 的数量 |
| 小队管理 | 列表显示当前配置，支持删除 [✕] 单个类型 |
| 启动仿真 | 点击 Start 按钮加载选定场景并生成配置的 Agent |

## UI 架构

```
MASetupHUD
└── MASetupWidget (Slate)
    ├── Header: "Multi-Agent Setup"
    ├── Scene Selection
    │   └── SComboBox<FString> (场景下拉)
    ├── Add Agent Section
    │   ├── SComboBox<FString> (Agent 类型下拉)
    │   └── Add Button (蓝色)
    ├── Squad List
    │   └── SListView<FAgentEntry>
    │       └── [+] [-] [✕] 按钮组
    └── Footer
        ├── Clear Button (红色)
        └── Start Button (绿色)
```

## 数据流

```
MASetupWidget (用户配置)
    ↓ OnStartClicked()
MAGameInstance::SetupConfig (存储配置)
    ↓ OpenLevel(SelectedScene)
MAGameMode::BeginPlay()
    ↓ 读取 SetupConfig
MAAgentManager::SpawnAgentByType()
    ↓ 按配置生成 Agent
```

- 数量徽章: 圆角背景显示

## 按钮映射机制

为支持动态列表中的按钮事件，使用 `TMap` 存储按钮到索引的映射：

```cpp
TMap<TSharedPtr<SButton>, int32> IncreaseButtonIndexMap;  // [+] 按钮
TMap<TSharedPtr<SButton>, int32> DecreaseButtonIndexMap;  // [-] 按钮
TMap<TSharedPtr<SButton>, int32> RemoveButtonIndexMap;    // [✕] 按钮
```

刷新列表时清空映射，重新绑定回调。

## 使用方式

1. 启动编辑器，打开 Setup 地图
2. 运行游戏进入 Setup UI
3. 选择场景
4. 添加 Agent 类型和数量
5. 点击 Start 进入仿真
